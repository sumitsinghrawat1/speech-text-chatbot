# Voice-Based AI Assistant for Visually Impaired

## Overview
This project is an AI-powered voice assistant developed to help visually impaired individuals perform daily tasks using voice commands. The system enables hands-free interaction through speech recognition and text-to-speech technology.

## Features
- Voice command interaction
- Speech-to-text conversion
- Text-to-speech responses
- Weather updates
- News updates
- Music playback
- Task assistance

## Technologies Used
- Python
- Speech Recognition
- Text-to-Speech (TTS)
- Tkinter
- gTTS

## Purpose of the Project
The main objective of this project is to improve accessibility and assist visually impaired users in performing daily activities more easily through voice-based interaction.

## Future Improvements
- AI-powered conversation system
- Smart home integration
- Multiple language support
- Improved voice recognition accuracy
